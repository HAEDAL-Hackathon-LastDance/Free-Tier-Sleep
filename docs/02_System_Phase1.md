# [기획서 2] 시스템 로직: Phase 1 (도주 - 번지 몽유병)

## 1. 핵심 메커니즘: 장력 물리 엔진 (The Tether)
*   **개요:** 캐릭터가 시작점(서버)에서 멀어질수록 강하게 뒤로 당겨지는 플랫포머.
*   **Unity 구현 로직:**
    *   캐릭터와 시작점 앵커(Anchor)에 `Rigidbody2D` 부착.
    *   두 객체를 `SpringJoint2D`로 연결.
    *   **Distance 튜닝:** `Distance` 값을 0으로 설정하여 항상 시작점 위치로 돌아가려는 힘을 받도록 설계.
    *   **Frequency & Damping:** `Frequency`(진동수)를 동적으로 조절. 캐릭터가 X축으로 맵의 50%를 넘어가면 `Frequency`를 올려 장력을 기하급수적으로 증가시킴.

## 2. 플레이어 조작 (Input)
*   좌우 이동(`Horizontal`) 및 점프(`Jump`).
*   **조작의 불편함 극대화:** 장력 때문에 공중에서 점프 궤도가 포물선이 아닌 뒤로 휘어지는 형태가 됨. 착지 타이밍을 계산하기 어렵게 만들어 의도된 피로도 제공.

## 3. 레벨 기믹 (장애물)
*   **스팸 팝업 발판 (Spam Platforms):** 
    *   허공에 떠 있는 [X(닫기)] 버튼이나 광고 배너 형태의 발판.
    *   밟으면 0.5초 뒤에 사라짐 (`Destroy` 또는 `SetActive(false)` 처리).
*   **알림창 탄환 (Notification Projectiles):**
    *   화면 밖에서 날아오는 카톡, 이메일 알림창들.
    *   피격 시 캐릭터를 뒤(시작점 방향)로 강하게 넉백시킴 (넉백 벡터에 `SpringJoint2D`의 당기는 힘이 더해져 치명타가 됨).

## 4. 진행도 및 실패 조건
*   **진행도 UI:** 화면 상단에 '다운로드 진행률' 바를 배치. 문에 가까워질수록 99%에서 멈춰있는 등 킹받는 UI 연출.
*   **실패:** 뒤로 튕겨 날아가 화면 좌측 경계(Dead Zone) 밖으로 떨어지면 처음부터 다시 시작.
