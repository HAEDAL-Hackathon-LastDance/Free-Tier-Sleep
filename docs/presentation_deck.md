# 🎮 [무료 수면 요금제 (Free-Tier Sleep)] 24슬라이드 피칭 덱 & 발표 대본 (v6)

본 대본은 대회 공식 주제 키워드를 초입부에 강조하고, 인덱스 문자`(1/3 등)`를 제거하여 자연스러운 흐름을 연출한 **총 24슬라이드 구조**의 최종 발표 양식입니다.

---

*   **핵심 키워드 하이라이팅:** **연결**, **불편**, **경계**, **꿈**, **과잉** 키워드는 슬라이드에서 **초록색 (Neon Green)**으로 돋보이게 처리되었습니다.
*   **폰트:** 제목 `Pretendard SemiBold` | 본문 `Pretendard Thin`
*   **이미지 비율:** 16:9 와이드 (가로 넓이 보정)

### [Slide 2] 대회 주제: 키워드 5종의 시스템적 융합 설계
*   **이미지 없음 (텍스트 전용 집중 슬라이드)**
*   **내용 (초록색 포인트 강조):**
    *   ■ **연결 (Connection)** : 뇌파를 클라우드 시스템에 상시 동기화하여 수면을 강제 통제받는 디스토피아 미래 기술 설정.
    *   ■ **불편 (Inconvenience)** : 요금을 낼 돈이 없어 밟아야 하는 스팸 광고창 닫기 `[X]` 버튼 발판 기믹과 조작 방해 요소 설계.
    *   ■ **경계 (Boundary)** : 과부하 네트워크를 단절하고 무의식의 심연인 오프라인 경계를 탈옥(Jailbreak)해 넘어가려는 목표.
    *   ■ **꿈 (Dream)** : 잠드는 순간부터 가동되는 Nightmare.exe 방화벽 강제 징수 프로그램과의 공허 구역 디펜스 대립.
    *   ■ **과잉 (Excess)** : 쏟아지는 에너미 객체 연산 부하에 비례해 마우스 감도 렉(조작 지연)을 강제 발생시켜 과잉 데이터 체감.
*   **발표 대본:**
    > "본 프로젝트는 해커톤 주제인 5가지 키워드를 유기적으로 조합했습니다. 특히 **연결**, **불편**, **경계**, **꿈**, **과잉**이라는 키워드들이 단순히 텍스트 시나리오에 머물지 않고 게임 내 시스템과 조작 물리로 어떻게 융합되었는지를 주목해 주시기 바랍니다."
